STAR-GET

Demostração moderna de cadastro biométrico e simulação de porta com reconhecimento facial

Cadastrar Testar reconhecimento Simulação da porta

Rápido resumo

Este frontend usa face-api.js para reconhecimento no cliente (requer que os modelos estejam disponíveis em /models/). Caso não queira usar modelos, o sistema tem fallback por hash para demonstração.

  • Cadastro seguro: descritores gravados localmente (IndexedDB/localStorage).
  • Teste de reconhecimento no navegador.
  • Simulação de porta com som e animação ao liberar o acesso.

Preparação

Suba a pasta /models/ com os modelos do face-api (ou use fallback). Posso gerar os modelos e instruções se quiser.

Hospede no GitHub Pages (site estático) — tudo roda no navegador.

Feito para demonstração — se quiser eu monto também um Docker e backend real.
